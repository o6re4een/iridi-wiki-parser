# JavaScript (iRidium script)

При разработке скриптов важно учитывать окружение \- сервер это, или интерфейс визуализации, т\.к\. окружение определяет набор доступных методов и способы обращения к данным: при переносе между интерфейсом и сервером, могут потребоваться изменения в скриптах\.

Для многих задач есть [готовые решения ](https://iridi.com/download/software/v3/#tab2), которые вы сможете использовать **не имея навыков программирования** \.

| [Изучите готовые решения ](https://iridi.com/download/software/v3/#tab2)и [форум поддержки ](https://support.iridiummobile.net/communities/18-en-international) возможно, ваша задача уже решена |

| [Пройдите обучение ](https://iridi.com/academy/) в Академии iRidium на курсах по iRidium script                                                                                                  |
| [Используйте руководство ](https://dev.iridi.com/JS_Guide) по JavaScript в iRidium                                                                                                               |
| [Используйте справочник ](https://dev.iridi.com/JS_Handbook) методов, свойств и событий i3 pro и iRidium server                                                                                  |
